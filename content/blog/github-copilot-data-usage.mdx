# Hook Intro
GitHub has announced that it will start using Copilot data from all user tiers to train and improve their AI models with automatic opt-in, raising concerns about user privacy.

# TL;DR
<Callout title="TL;DR" type="info">
* GitHub will use Copilot data from all user tiers to train AI models
* Users can opt out of data collection in their settings
* The data will be used to improve model performance and deliver more accurate code suggestions
* The change will take effect on April 24
</Callout>

## What Is GitHub Copilot and Why It Matters
GitHub Copilot is an AI-powered tool that helps developers write code faster and more efficiently. It uses machine learning algorithms to suggest code completions, refactor code, and even write entire functions. The tool has been widely adopted by developers, but the recent announcement has raised concerns about user privacy.

## Technical Deep Dive
The Copilot data that GitHub will collect includes inputs, outputs, code snippets, and associated context. This data will be used to train and improve the AI models, allowing them to better understand development workflows and deliver more accurate code suggestions. The data collection will be automatic, but users can opt out in their settings.

## Real-World Examples
Companies like Microsoft have already seen improvements in their AI models by incorporating real-world interaction data. For example, Microsoft has reported increased acceptance rates in multiple languages after incorporating interaction data from their employees.

## What the Community Is Saying
The community is divided on the issue, with some developers expressing concerns about user privacy and others seeing it as a necessary step to improve the AI models. On Reddit, some users have expressed frustration with the automatic opt-in, while others have praised GitHub for being transparent about the data collection.

## How to Get Started
To opt out of the data collection, follow these steps:
1. Go to your GitHub settings
2. Click on "Privacy"
3. Disable the option to allow GitHub to collect interaction data
