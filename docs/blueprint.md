# **App Name**: SecureAutomate

## Core Features:

- Hero Section with Comparison: Visually hook visitors by comparing 'AI Agency' prompts to custom code. Includes clear CTAs.
- Differentiation Table: Interactive table comparing custom automation to ChatGPT/no-code tools, highlighting key differences and benefits.
- Security-First Process Display: Showcase a 4-step security process (Threat Modeling, Secure Architecture, Testing, Monitoring) with expandable details for each step.
- Case Studies Showcase: Display anonymized case studies highlighting client type, problem, solution, and key metrics with a 'View Full Case Study' modal.
- Services Overview: Overview of services offered with icons, descriptions, and examples of AI-Powered Workflow Automation, System Integration and API Development, and Custom AI Chatbots & Support Systems.
- ROI Calculator: Interactive ROI calculator allowing prospects to input hourly rate, hours on manual tasks, and number of employees to estimate potential savings.
- Lead Qualifying Chatbot: An AI chatbot that engages website visitors, answers initial questions, and qualifies leads based on pre-defined criteria, such as industry and project size, acting as a tool to help prioritize potential clients, integrated to cal.com.

## Style Guidelines:

- Primary color: Deep ocean blue (#1A73E8) to convey security and trust.
- Background color: Light gray (#F5F5F5) for a clean and professional look.
- Accent color: Teal (#008080) to highlight key interactive elements and CTAs.
- Body and headline font: 'Inter' for a modern, neutral, and readable style.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use consistent and professional icons from 'lucide-react' related to security, automation, and AI.
- Use subtle animations via 'framer-motion' for section transitions, card hovers, and calculation updates to enhance user experience.
